# Project Management Package

A comprehensive project management solution with integrated time tracking and invoicing capabilities.

## Features

- **Project Management**
  - Create and manage multiple projects
  - Track project status and progress
  - Assign tasks and set deadlines
  - Manage project resources

- **Time Tracking**
  - Log time entries for tasks and projects
  - Track billable and non-billable hours
  - Generate time reports
  - Monitor team productivity

- **Invoicing**
  - Create professional invoices
  - Automatic invoice generation based on tracked time
  - Multiple currency support
  - Track payment status
  - Export invoices in various formats

## Installation

```bash
composer require prasso/project-management
```
